# Frontend Hackathon 2025 - Website Clone

This project was developed during a **2-hour Frontend Hackathon**. The challenge was to accurately recreate a provided website design using only pure HTML and CSS within a strict time limit.

## üöÄ Project Overview

The goal of this project was to demonstrate speed, accuracy, and proficiency in frontend layout techniques. It features a responsive design centered around a product showcase (E-commerce theme).

### Features
- **Pure HTML5 & CSS3**: Built without external frameworks to showcase core layout skills.
- **Product Gallery**: Includes a showcase of various products (shoes) with asset integration.
- **UI Elements**: Navigation bar, shopping cart integration, and user profile sections.
- **Rapid Development**: Entire structure and styling completed within a 120-minute window.

## üìÅ Repository Structure

- `index.html`: The main entry point containing the website structure.
- `style.css`: All styling, including layout, typography, and color schemes.
- `assets/`: (Images like `shoe_1.png`, `cart.png`, etc.) UI components and product imagery.
- `Document.html`: Supplementary project documentation or additional pages.
